# MC World Explorer 项目路线图

## 项目定位

MC World Explorer 是一个独立于 Minecraft 客户端运行的存档浏览工具。

核心目标：

在不启动 Minecraft 的情况下快速浏览、管理和预览世界存档。

最终目标：

- 快速查看存档信息
- 浏览世界地图
- 定位建筑与基地
- 三维世界浏览
- 自动巡航展示
- 动态壁纸模式

# V0.1 存档管理器

## 目标

完成最小可用产品（MVP）。

## 功能

### 存档扫描

扫描 Minecraft 存档目录。

### 存档列表

显示：

- 世界名称
- Minecraft版本
- 游戏模式
- 最后游玩时间
- 世界图标

### 存档详情

显示：

- 世界名称
- 创建时间
- 最后游玩时间
- 游戏时间
- 种子
- 出生点坐标
- 玩家坐标

## 完成标准

能够正常扫描并显示存档信息。

# V0.1.1 代码质量加固

## 目标

基于 V0.1 代码审查发现的问题，进行代码结构优化、基础设施补充和存档解析兼容性加固。

保持正常存档既有展示逻辑；允许修复已经确认的识别和兼容性缺陷。

## 重点

- 引入日志框架（SLF4J + Logback）
- 统一错误处理策略
- 提取 GameType 枚举
- 清理死代码
- 代码规范整理
- 补齐 Gradle Wrapper 和 Windows 启动脚本
- 修复大型整合包 `level.dat` 读取额度不足的问题
