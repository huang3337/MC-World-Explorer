# V0.2 世界缩略图开发进度记录

## 当前阶段

V0.2 处于开发前准备阶段，尚未正式开始。

本记录用于固定进入 V0.2 前的基线、范围和确认门。开发者明确确认前，不编写 Region 解析、缩略图生成、缓存或预览界面代码。

## 版本目标

在不启动 Minecraft、不依赖外部程序且不修改存档的前提下，生成并显示主世界静态俯视缩略图。

已确认的主要产品目标：

- 中心点依次使用主世界个人重生点、世界出生点、`(0, 0)`。
- 范围为 `1024 x 1024` 方块，输出为 `512 x 512` PNG。
- 使用方块地图颜色和高度明暗表现地形。
- 在程序内提供“世界预览”视图，并支持导出 PNG。
- 缓存、日志和导出文件全部保存在程序根目录。
- 不生成下界或末地缩略图。

完整设计见 `docs/superpowers/specs/2026-07-14-v0.2-world-thumbnail-design.md`。

## 正式开发顺序

1. 严格便携路径和日志迁移。
2. 缩略图中心点解析。
3. Region 文件头、区块定位和解压。
4. 现代区块 Section、Palette 和地表采样。
5. 地图颜色、高度明暗和 PNG 输出。
6. 缓存与失效判断。
7. JavaFX 后台生成和内置预览。
8. PNG 导出和目录错误处理。
9. 自动化测试、真实存档只读验收和整体审查。
10. 文档同步和 Windows x64 便携包验收。

每一步单独实施、单独测试和单独复核，不混入 V0.3 功能。

## 禁止提前开发

- 下界和末地缩略图。
- 地图拖动、连续缩放、瓦片浏览和坐标跳转。
- 地图标记系统。
- Mod 资源包或纹理加载。
- 三维浏览、自动巡航和动态壁纸。
- 任何 Minecraft 存档写入功能。
